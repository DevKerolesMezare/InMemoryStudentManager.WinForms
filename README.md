# InMemoryStudentManager.WinForms

A simple Windows Forms application for managing students in memory using C#.

The application allows adding, editing, and deleting students using an in-memory list without a database connection.

## Features

- Add Students
- Edit Students
- Delete Students
- In-Memory Data Storage
- Basic CRUD Operations
- Windows Forms UI

## Technologies Used

- C#
- .NET Framework
- WinForms
- OOP
- List Data Structure

## Purpose

This project was built to practice basic CRUD operations, form handling, and object-oriented programming concepts in C#.
